\f0\fs24 \cf0 # Copilot for Mac\
\
**Copilot for Mac** is a lightweight Electron wrapper around the Microsoft Copilot web interface.  \
It gives you a dedicated desktop app with macOS-style menus and a few extra tools:\
\
- Reload, Zoom In/Out, Fullscreen, Always on Top\
- Save page as **PDF** or **Text**\
- Take a **screenshot** of the window\
- Copy the current page URL or open it in your default browser\
- A built-in **Notes** window (stored locally in the app)\
- Global shortcut to show/hide the window\
\
> This is an unofficial wrapper. It simply loads the official Copilot site in an Electron shell.\
\
## Features\
\
- 
\f1 \uc0\u9989 
\f0  Dedicated Copilot window for macOS  \
- 
\f1 \uc0\u9989 
\f0  macOS-style menu bar (File, Edit, View, Window, Help)  \
- 
\f1 \uc0\u9989 
\f0  Notes window with persistent local storage  \
- 
\f1 \uc0\u9989 
\f0  Screenshot, PDF, and text export of the current page  \
- 
\f1 \uc0\u9989 
\f0  Global shortcut (e.g., Ctrl+Alt+G) to toggle visibility  \
\
## Requirements\
\
- Node.js (LTS is recommended)\
- npm\
- macOS\
\
